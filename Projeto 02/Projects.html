Calculadora

Calculadora simples, dispõe das quatro operações básicas. É possivel resolver problemas matemáticos grandes com vários algarismos , limpar o ultimo número digitado, limpar toda a tela e ate mesmo "desligar" a calculadora.(Projeto Responsivo).

Calculadora de IMC

Calculadora de IMC(Índice de massa corporal) simples. Há uma tabela com os dados para consulta, e é possivel saber se está abaixo do peso, com peso ideial, acima do peso e até com obesidade.(Projeto não responsivo).

Urna eletronica

Projeto da Urna, contém todas as funcionalidades previstas para uma urna, possui candidatos cadastrados. É possivel votar somente uma vez(afinal, cada pessoa vota somente uma vez não é? kkk), após votar, recarregue a página.(Projeto não responsivo)
